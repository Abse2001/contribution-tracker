# Changelog 2026-04

- Copper pours can now be flagged as **unbroken** to prevent isolation gaps ([props #628](https://github.com/tscircuit/props/pull/628), [core #2129](https://github.com/tscircuit/core/pull/2129))
- Nets / traces accept **routingPhaseIndex** for phased autorouting ([props #629](https://github.com/tscircuit/props/pull/629), [core #2139](https://github.com/tscircuit/core/pull/2139))
- Autorouter adds **escape-via stage** + via-to-via clearance enforcement ([autorouter #875](https://github.com/tscircuit/tscircuit-autorouter/pull/875), [#874](https://github.com/tscircuit/tscircuit-autorouter/pull/874))
- Footprinter: comprehensive **courtyard geometry fixes** for SOIC, JST, BGA, DIP, axial & more ([footprinter #590](https://github.com/tscircuit/footprinter/pull/590), [#589](https://github.com/tscircuit/footprinter/pull/589), [#591](https://github.com/tscircuit/footprinter/pull/591), [#593](https://github.com/tscircuit/footprinter/pull/593), [#594](https://github.com/tscircuit/footprinter/pull/594))
- SVG & Canvas now render **round line-joins** for PCB courtyards ([circuit-to-svg #538](https://github.com/tscircuit/circuit-to-svg/pull/538), [circuit-to-canvas #220](https://github.com/tscircuit/circuit-to-canvas/pull/220))
- PCB Viewer: drill layer always on top, inner-layer pours render, via-clearance errors highlighted, toolbar clicks no longer block panning ([pcb-viewer #735](https://github.com/tscircuit/pcb-viewer/pull/735), [#737](https://github.com/tscircuit/pcb-viewer/pull/737), [#738](https://github.com/tscircuit/pcb-viewer/pull/738), [#729](https://github.com/tscircuit/pcb-viewer/pull/729))
- EasyEDA import keeps **{NAME}** placeholders & fixes model placement + oval/pill plated holes ([easyeda-converter #374](https://github.com/tscircuit/easyeda-converter/pull/374), [#375](https://github.com/tscircuit/easyeda-converter/pull/375), [#377](https://github.com/tscircuit/easyeda-converter/pull/377))
- KiCad pipeline: inner-layer & arc support on import ([kicad-to-circuit-json #54](https://github.com/tscircuit/kicad-to-circuit-json/pull/54), [#53](https://github.com/tscircuit/kicad-to-circuit-json/pull/53)); rotated pill-hole NPTH export ([circuit-json-to-kicad #195](https://github.com/tscircuit/circuit-json-to-kicad/pull/195)); STEP fallback for missing 3D models ([kicad-mod-cache #21](https://github.com/tscircuit/kicad-mod-cache/pull/21))
- Circuit-JSON adds **layer** property for pcb_note* with full rendering support ([circuit-json #529](https://github.com/tscircuit/circuit-json/pull/529), [core #2121](https://github.com/tscircuit/core/pull/2121), [circuit-json-util #91](https://github.com/tscircuit/circuit-json-util/pull/91))
- STEP exporter delivers face-level colours & style reuse ([circuit-json-to-step #71](https://github.com/tscircuit/circuit-json-to-step/pull/71))
- Canvas viewer shows **translucent inner-layer copper pour colours** ([circuit-to-canvas #228](https://github.com/tscircuit/circuit-to-canvas/pull/228))
- CLI: new `tsci check trace-length` and configurable worker timeout ([cli #2633](https://github.com/tscircuit/cli/pull/2633), [#2667](https://github.com/tscircuit/cli/pull/2667))
- USB-C: inner schematic symbol, standardised pin labels, and no more false missing-footprint errors ([core #2118](https://github.com/tscircuit/core/pull/2118), [#2131](https://github.com/tscircuit/core/pull/2131), [#2136](https://github.com/tscircuit/core/pull/2136))
- Connectors: schematic layout fixed & CAD model retained when fetching parts ([core #2134](https://github.com/tscircuit/core/pull/2134), [#2135](https://github.com/tscircuit/core/pull/2135))
- Copper-pour obstacles now recognise **vias as conductors** ([core #2140](https://github.com/tscircuit/core/pull/2140))
- Full support for **rotated pill-hole with rect pad** plated holes across toolchain ([core #2141](https://github.com/tscircuit/core/pull/2141), [circuit-json-to-gltf #156](https://github.com/tscircuit/circuit-json-to-gltf/pull/156))
- CAD Component Viewer: drag-and-drop model loading + hierarchy & material colours preserved ([cad-component-viz #5](https://github.com/tscircuit/cad-component-viz/pull/5), [#7](https://github.com/tscircuit/cad-component-viz/pull/7))
- Graphics Debug: click any canvas object to inspect its label ([graphics-debug #110](https://github.com/tscircuit/graphics-debug/pull/110))
