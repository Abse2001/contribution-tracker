# Changelog 2026-04

- Added “unbroken” copper-pour option (props [#628](https://github.com/tscircuit/props/pull/628) & core [#2129](https://github.com/tscircuit/core/pull/2129))
- Autorouter: new escape-via stage + scoring, inner-layer pour support, via-clearance & route-stitching fixes (autorouter [#875](https://github.com/tscircuit/tscircuit-autorouter/pull/875), [#880](https://github.com/tscircuit/tscircuit-autorouter/pull/880), [#876](https://github.com/tscircuit/tscircuit-autorouter/pull/876), [#886](https://github.com/tscircuit/tscircuit-autorouter/pull/886), [#854](https://github.com/tscircuit/tscircuit-autorouter/pull/854), [#842](https://github.com/tscircuit/tscircuit-autorouter/pull/842))
- Footprinter: switched to generic body-based stepped courtyards and fixed KiCad-parity dimensions across many packages (footprinter [#590](https://github.com/tscircuit/footprinter/pull/590), [#591](https://github.com/tscircuit/footprinter/pull/591), [#594](https://github.com/tscircuit/footprinter/pull/594))
- EasyEDA converter: kept {NAME} placeholder, corrected CAD placement, parsed undefined tokens & pill/oval plated holes (easyeda-converter [#374](https://github.com/tscircuit/easyeda-converter/pull/374), [#375](https://github.com/tscircuit/easyeda-converter/pull/375), [#376](https://github.com/tscircuit/easyeda-converter/pull/376), [#377](https://github.com/tscircuit/easyeda-converter/pull/377))
- KiCad importer: inner-layer & arc support plus accurate oval-drill and solder-paste handling (kicad-to-circuit-json [#54](https://github.com/tscircuit/kicad-to-circuit-json/pull/54), [#53](https://github.com/tscircuit/kicad-to-circuit-json/pull/53), [#58](https://github.com/tscircuit/kicad-to-circuit-json/pull/58), [#59](https://github.com/tscircuit/kicad-to-circuit-json/pull/59))
- pcb_note elements now carry a layer attribute for correct rendering (circuit-json [#529](https://github.com/tscircuit/circuit-json/pull/529), circuit-json-util [#91](https://github.com/tscircuit/circuit-json-util/pull/91), core [#2121](https://github.com/tscircuit/core/pull/2121))
- SVG/Canvas: round line-joins for courtyards and auto-mirrored bottom-copper text (circuit-to-canvas [#220](https://github.com/tscircuit/circuit-to-canvas/pull/220), [#226](https://github.com/tscircuit/circuit-to-canvas/pull/226); circuit-to-svg [#538](https://github.com/tscircuit/circuit-to-svg/pull/538))
- PCB Viewer UX: block toolbar hit-tests, group/collapse error list, keep drill layer on top (pcb-viewer [#729](https://github.com/tscircuit/pcb-viewer/pull/729), [#733](https://github.com/tscircuit/pcb-viewer/pull/733), [#735](https://github.com/tscircuit/pcb-viewer/pull/735))
- CAD Component Viewer: drag-and-drop model loading & full hierarchy preservation (cad-component-viz [#5](https://github.com/tscircuit/cad-component-viz/pull/5), [#7](https://github.com/tscircuit/cad-component-viz/pull/7))
- STEP export: face-level color styling & style reuse (circuit-json-to-step [#71](https://github.com/tscircuit/circuit-json-to-step/pull/71))
- Graphics Debug: convert any graphics object to PNG (graphics-debug [#109](https://github.com/tscircuit/graphics-debug/pull/109))
- DRC: ignore courtyard-overlap between different layers (checks [#130](https://github.com/tscircuit/checks/pull/130))
- CLI: added `tsci check trace-length` command (cli [#2633](https://github.com/tscircuit/cli/pull/2633))
- KiCad exporter: rotated pill NPTH hole support (circuit-json-to-kicad [#195](https://github.com/tscircuit/circuit-json-to-kicad/pull/195))
