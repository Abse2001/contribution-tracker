- Copper pours: new optional `unbroken` flag (props #628, core #2129) and `isCopperPour` obstacle flag for autorouter (#840)
- Autorouter: added Escape-Via stage (#875), via-clearance checks (#854), HighDensityForceImproveSolver integration (#876), path-simplification fixes (#870) and PNG stage dumps (#848); numerous perf tweaks (max-node ratio, node pruning, availableZ support)
- Footprinter: switched all major packages to generic body-based courtyards (PRs #584-#591, #593-#594) and fixed SOIC/other dims to KiCad standards
- Courtyard rendering: round line-join support in SVG/Canvas (circuit-to-svg #538, circuit-to-canvas #220)
- EasyEDA converter: preserved {NAME} placeholders & robust CAD model placement + oval-hole support (PRs #374-#377, #379)
- KiCad I/O: inner-layer & arc import, pill-hole alignment fixes (kicad-to-circuit-json #54, #53, #58, #60; circuit-json-to-kicad #195, #200)
- STEP/3D: new face-level coloring & style-reuse pipeline (circuit-json-to-step #71); 3D viewer perf/cleanup (#758-#760)
- CLI: added `tsci check trace-length`, configurable worker time-out, and rollup fix (cli #2633, #2667, #2665)
- High-density repair toolchain: dataset-oriented benches, dynamic-margin nudging, interior-point clamping, new solver & metrics (high-density-repair02 PRs #37-#47; repair01 #3-#4)
- Tiny-Hypergraph: bus selector, centerline/region routing & CM5IO data fixes (#39-#50)
- Core: routingPhaseIndex support (#2139 + props #629), vias counted in obstacle connectivity (#2140), USB-C symbol/layout fixes (#2131, #2134-#2136) and connector orientation variants (#2138)
- PCB Viewer: grouped error list, via-clearance highlight, inner-layer pour render fix (#733, #738, #737)
- CAD Component Viz: drag-and-drop model loader & full hierarchy preservation (#5-#7)
- Graphics-Debug: click-to-inspect canvas objects (#110)
- Misc: 3D model STEP fallback for KiCad libs (kicad-mod-cache #21); translucent inner-layer pour colors & mirrored bottom-copper text (circuit-to-canvas #226, #228)
